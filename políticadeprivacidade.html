Política de Privacidade

Este termo visa informar e obter a ciência dos usuários do site sobre a coleta, utilização e o compartilhamento de seus dados pessoais em conformidade com a legislação vigente, incluindo a Lei Geral de Proteção de Dados Pessoais (LGPD – Lei nº 13.709/2018).

1. Finalidade da Coleta de Dados

Os dados pessoais fornecidos pelos usuários serão coletados e utilizados exclusivamente para:

  • Responder a consultas realizadas através do site;
  • Agendar atendimentos ou consultas jurídicas;
  • Enviar informações sobre serviços prestados pelo escritório;
  • Compartilhar conteúdo jurídico de interesse, caso autorizado pelo usuário.
  • Cumprir obrigações legais ou regulatórias.

2. Tipos de Dados Coletados

Os dados coletados podem incluir, mas não se limitam a:

  • Nome completo;
  • Endereço de e-mail;
  • Telefone de contato;
  • CPF ou outros dados para identificação (se necessário para um serviço específico);
  • Outras informações fornecidas voluntariamente pelo usuário através dos formulários do site.

    • Esses dados podem ser coletados diretamente através de formulários no site, e-mails ou outras formas de comunicação.

3. Compartilhamento de Dados

Os dados pessoais fornecidos pelos usuários não serão compartilhados com terceiros, salvo nas seguintes hipóteses:

  • Para cumprimento de obrigações legais ou regulações;
  • Com consentimento expresso do usuário;
  • Quando necessário para execução de serviços contratados pelo usuário.

4. Direitos do Usuário

Em conformidade com a LGPD, o usuário tem direito a:

  • Acessar os dados pessoais fornecidos;
  • Solicitar a correção de dados incorretos, incompletos ou desatualizados;
  • Solicitar a exclusão de seus dados pessoais, exceto quando a manutenção for exigida por lei;
  • Revogar o consentimento para tratamento de dados pessoais, quando aplicável.

    • Para exercer qualquer um desses direitos, entre em contato através do e-mail: leticia.advog@

5. Segurança de Dados

Adotamos medidas técnicas e organizacionais adequadas para proteger seus dados pessoais contra acesso não autorizado, perda, alteração ou divulgação indevida. No entanto, nenhum sistema é totalmente seguro, e não podemos garantir a segurança absoluta das informações transmitidas pelo site.

6. Cookies e Tecnologias Semelhantes

Utilizamos cookies para melhorar sua experiência no site. Esses cookies coletam informações como:

  • Dados de navegação (páginas acessadas, tempo no site, etc.)
  • Preferências do usuário.

    • Você pode ajustar as configurações do seu navegador para bloquear ou excluir cookies, mas isso pode afetar a funcionalidade do site.

7. Retenção de Dados

Reteremos seus dados pessoais apenas pelo tempo necessário para cumprir as finalidades descritas nesta política ou conforme exigido por lei.

8. Alterações nesta Política de Privacidade

Esta política pode ser atualizada periodicamente. Recomendamos que você revise este documento regularmente para se manter informado sobre como estamos protegendo seus dados. A data da última atualização foi: 21 de janeiro de 2025.
